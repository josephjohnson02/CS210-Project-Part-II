# CS210-Project-Part-II: Palette Maker
## Project Description
My project is a “palette maker” made with Java, using JavaFX for its GUI. The program is intended for web/graphic designers to create aesthetically appealing palettes without having to know too much color theory.

Palette Maker is a desktop application that's fast and easy to use. Its interface is simple to understand and effective at creating and modifying palettes for the user, using algorithms to simulate harmony rules of color theory such as complementary, analogous and triadic schemes to ensure they’re aesthetically coherent.

Users are able to select a color, are given multiple options for other colors they can use with it, and can choose which ones to add until they’ve created a palette that is satisfactory for whatever its purpose may be.

Initial Feature List:
- Select a base color using a color picker
- Generate visuals for matching color suggestions
- Add colors to a palette
- Remove colors from a palette
- Display the palette visually
- Refresh color suggestions based on current palette
- Copy HEX values from colors in palette by clicking them
- Save palettes to files
- Load palettes from files
